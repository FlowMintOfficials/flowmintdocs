FlowMint Recorder

Record browser tests, add assertions, and export to Playwright (JavaScript), Python, Java, C#, or Ruby. Data-driven tests (CSV/JSON), test suites, and multi-language export in one Chrome extension.

  • Record — Clicks, typing, navigation, selects, hover, scroll. Smart selectors (data-testid, id, ARIA).
  • Assertions — Text, visibility, URL, attribute, element count. From side panel or floating toolbar.
  • Export — JavaScript (Playwright), Python, Java, C#, Ruby. Copy or download.
  • Data-driven — CSV/JSON data; run test once per row; variable interpolation.
  • Test suites — Save tests, run later, rename. Organize in the Suites tab.

About FlowMint

FlowMint builds tools for developers and QA teams. FlowMint Recorder is our first product — a Chrome extension that lets you record browser tests and export them to the language and framework you use. All data stays on your device; we don’t collect or transmit your tests.
